Le langage HTML

Le code HTML est basé sur la séparation entre le "fond" et la "forme" que peut prendre l'information à transmettre.

Le fond concerne :

  • le texte
  • carre
  • les titres
  • les images et autres types de contenus
  • les liens vers d'autres pages
  • ...

  1. la couleur du texte
  2. la couleur du fond de la page
  3. la couleur
  4. la police de caractère
  5. les enrichissements de texte
  6. la division de la page en sections (menu, en-tête...)
  7. le positionnement des différents contenus
  8. ...

"le langage html est simple est facille à apprendre cliquer ici pour dowland un résumer complet de ce langage."

Le langage html ne permet que dans un premier temps de s'occuper du fond.
La forme sera gérée par un second fichier que nous verrons ultérieurement.


ATTENTION

Le html est seulment pour écrire la structure du site. Pour le visuelle on utilise le CSS il permet de mettre des couleur et de donné du style a ton site web.
